# telegram-raid-botnet

## Что нужно сделать перед запуском?
- Установить модули

`pip3 install -r requirements.txt`

- Добавить аккаунты

`cd sessions`<br>
`python3 add_session.py`

- Добавить файлы в папку `media`
- Отредактировать `config.toml`
